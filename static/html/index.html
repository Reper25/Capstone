Home Page Route Links

Users
Method/s Path Description
GET /Users Gets all the users from the Users Table
POST /User/:id Gets an individual User
DELETE /User/:id Delete's a user using the userId
PUT /User:id Updates a user in the Users table using their userId
Products
Method/s Path Description
GET /Products Gets all the Products from the products Table
POST /Product/:prodId Gets an individual product
DELETE /Product/:prodId Delete's a product using the prodId
PUT /Product:prodId Updates a product in the Products table using their prodId
